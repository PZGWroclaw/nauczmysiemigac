# Nauczmy się migać – Kurs Polskiego Języka Migowego (PJM)

Prosta strona edukacyjna z lekcjami **Polskiego Języka Migowego** dla osób zaczynających naukę.

## 📚 O kursie

Ten projekt zawiera:
- Krótkie lekcje PJM
- Filmy edukacyjne
- Słowniki i materiały
- Ćwiczenia praktyczne

## 🎓 Moduły nauki

### Moduł 1 – Podstawy PJM
- Alfabet PJM
- Podstawowe zwroty: cześć, dziękuję, proszę, tak/nie
- Słowniczek PDF

### Moduł 2 – Komunikacja codzienna
- Przedstawianie się
- Pytania: kto, co, gdzie, kiedy
- Krótkie dialogi

### Moduł 3 – Sytuacje praktyczne
- W sklepie
- W przychodni
- W pracy / urzędzie
